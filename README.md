# CLOAK Wallet

> Privacy wallet for CLOAK on Telos

Private shielded transactions using zk-SNARKs on the Telos blockchain.

## Features

- **Shielded Transactions**: Full zk-SNARK privacy using the Sapling circuit
- **Deterministic Vaults**: Secure vault system with deterministic key derivation
- **View-Only Wallets**: Import Full Viewing Keys (FVK) for watch-only access
- **Multi-Platform**: Linux, macOS, Windows, Android, iOS
- **Fast Sync**: Table-based synchronization with step-based progress tracking
- **Self-Custody**: All information recoverable from seed phrase
